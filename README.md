# Satyam Kumar Portfolio

A clean and responsive personal portfolio website built to present profile details, technical skills, project highlights, education, certifications, and contact links in a professional single-page layout.

## Live Project Purpose

This portfolio is designed to:

- introduce Satyam Kumar with a strong first impression
- showcase technical strengths and project interests
- present education and internship background clearly
- make GitHub, LinkedIn, resume, email, and phone access easy

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Font Awesome
- Google Fonts

## Project Structure

```text
Portfolio/
|-- assets/
|   |-- css/
|   |   `-- style.css
|   |-- docs/
|   |   `-- resume.pdf
|   |-- images/
|   |   `-- profile.png
|   `-- js/
|       `-- main.js
|-- index.html
`-- README.md
```

## Features

- polished hero section with clear CTA buttons
- structured sections for about, skills, projects, education, and contact
- responsive layout for desktop and mobile screens
- reusable panels, chips, and cards for easier future updates
- scroll reveal effects and active navigation highlighting
- downloadable resume and direct contact actions

## How To Run Locally

1. Clone the repository:

```bash
git clone https://github.com/satyamkr-05/portfolio.git
```

2. Open the project folder.

3. Run `index.html` in your browser.

## Customization Guide

- Update profile details in `index.html`
- Replace the profile image in `assets/images/profile.png`
- Replace the resume in `assets/docs/resume.pdf`
- Adjust colors, spacing, and layout styles in `assets/css/style.css`
- Add more interactions in `assets/js/main.js`
